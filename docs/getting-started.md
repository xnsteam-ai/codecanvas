## <a id="installation"></a>üì• Installation

CodeCanvas can be installed on Windows, macOS, and Linux. Visit the [releases page](https://github.com/<YOUR_GITHUB_USERNAME>/codecanvas/releases) for installation instructions.

**Quick Install Options:**
- **Windows**: Use Winget with `winget install -e --id CodeCanvas.CodeCanvas`
- **macOS**: Use Homebrew with `brew install --cask codecanvas`
- **Linux**: Use Snap with `snap install codecanvas --classic`

## <a id="first-steps"></a>First Steps

After installing CodeCanvas, here are some first steps to get started:

1. **Open a folder**: Use File > Open Folder to open your project
2. **Install extensions**: Click on the Extensions icon in the sidebar to browse and install extensions
3. **Configure settings**: Use File > Preferences > Settings to customize your editor

## <a id="basic-usage"></a>Basic Usage

CodeCanvas works just like Visual Studio Code, with a few differences:

- It uses Open VSX for extensions by default instead of the Visual Studio Marketplace
- It doesn't include Microsoft telemetry or branding
- Some proprietary features may not be available

## <a id="keyboard-shortcuts"></a>‚å®Ô∏è Essential Keyboard Shortcuts

Here are some essential keyboard shortcuts to get you started:

- `Ctrl+P` (Windows/Linux) or `Cmd+P` (macOS): Quick Open, Go to File
- `Ctrl+Shift+P` (Windows/Linux) or `Cmd+Shift+P` (macOS): Show Command Palette
- `Ctrl+,` (Windows/Linux) or `Cmd+,` (macOS): User Settings
- `Ctrl+K Ctrl+S` (Windows/Linux) or `Cmd+K Cmd+S` (macOS): Keyboard Shortcuts

## <a id="recommended-extensions"></a>üß© Recommended Extensions

Get started faster with these essential extensions:

- **Python Development**: `ms-python.python` - IntelliSense, debugging, and formatting
- **JavaScript/TypeScript**: `ms-vscode.vscode-typescript-next` - Enhanced language support
- **Git Enhancement**: `eamodio.gitlens` - Supercharge Git capabilities
- **Code Formatting**: `esbenp.prettier-vscode` - Automatic code formatting
- **Popular Theme**: `dracula-theme.theme-dracula` - Eye-friendly dark theme

To install any extension:
1. Click the Extensions icon in the sidebar (üß©)
2. Search for the extension name
3. Click "Install"
